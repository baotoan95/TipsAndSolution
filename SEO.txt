SEO
- Công cụ:
	+ Web developer
	+ SEO Quake
	
- Các thông số SEO
	+ PR: Tầm quan trọng của website (Trang web khác trỏ tới trang mình và ngược lại)
	+ I (Google index): Tổng số trang con được Google lưu ở DB
	+ Alexa rank: thứ hạng của website đứng thứ hạng bao nhiêu đối với country and global
	+ Age: tuổi của domain
	+ Density: mật độ từ khóa
	+ Internal link: liên kết nội bộ
	+ External link: Trang khác trở tới trang hiện tại
- SEO quake diagnosis: check website chuẩn SEO
- Web developer:
	+ Kiểm tra heading
	+ Tối ưu hình ảnh (thẻ alt) alt phải chứa từ khóa
	
- Chú ý khi dev web:
	+ Hình ảnh phải có alt
	+ Từ khóa phải để trong heading tags
